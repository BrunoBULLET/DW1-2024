Tags de código

code

code é usado para envolver partes de texto que representam código de programação ou texto que deve ser interpretado literalmente, sem formatação especial. Quando o navegador encontra esse elemento, ele normalmente exibe o conteúdo dentro dele usando uma fonte mono-espaçada, como Courier, e geralmente mantém a formatação exata (espaços, quebras de linha) do texto contido dentro do elemento.

Por exemplo:

Olá, mundo

samp

O elemento samp em HTML é usado para envolver texto que representa a saída de um programa de computador ou de um script. "samp" é uma abreviação de "sample output" (saída de exemplo). Quando o navegador encontra esse elemento, ele normalmente exibe o conteúdo dentro dele usando uma fonte mono-espaçada, semelhante ao elemento code, mas com a intenção específica de indicar que o texto é uma saída de programa.

Por exemplo:

A saída do programa é: Olá, mundo!

pre

O elemento pre em HTML é usado para definir texto pré-formatado. Quando você envolve texto dentro de pre, o navegador exibe esse texto exatamente como está no código HTML, preservando espaços em branco, quebras de linha e qualquer outra formatação textual. O elemento pre é frequentemente utilizado para exibir blocos de código de programação, mensagens de terminal, dados tabulares e qualquer outra informação que dependa de formatação específica, onde a precisão do layout é importante.

Por exemplo:

A saída do programa é: 

Olá, mundo!

A tag kbd em HTML é usada para representar a entrada de teclado do usuário, como teclas individuais ou combinações de teclas. "kbd" é uma abreviação de "keyboard" (teclado). Quando o navegador encontra esse elemento, geralmente exibe o conteúdo dentro dele usando uma fonte monoespaçada, semelhante ao elemento code, para indicar visualmente que se trata de entrada de teclado.

Por exemplo:

Pressione Ctrl + C para copiar.
